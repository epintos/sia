Pasos para correr el trabajo practico:

1. mvn clean package assembly:single
2. mv target/TP1-SIA-0.0.1-SNAPSHOT-jar-with-dependencies.jar .
3. java -Xmx2g -jar TP1-SIA-0.0.1-SNAPSHOT-jar-with-dependencies.jar --help y seguir los parametros.

Es necesario ingresar boardsize o filename como parametros obligatorios para correr el trabajo.
